Online Shoe Store

E-commerce Project Documentation

Project Overview

This is an e-commerce website for selling shoes online. The project is built using PHP and MySQL, featuring user authentication, product management, shopping cart functionality, and an admin panel.

Key Features

  • User registration and authentication
  • Product browsing and searching
  • Shopping cart functionality
  • Secure checkout process
  • Admin panel for product management

Technologies Used

  • PHP
  • MySQL
  • HTML/CSS
  • JavaScript
  • Bootstrap
