Project Overview

This online shoe store is a complete e-commerce solution built with PHP and MySQL. It features user authentication, product browsing, shopping cart functionality, checkout process, and an admin panel for managing products and orders.

The project was developed by Swaroop and Adarsha as a demonstration of web development skills using PHP, MySQL, HTML, CSS, and JavaScript.

Installation

  1. Clone the repository from GitHub
  2. Import the db/alphaware.sql file into your MySQL database
  3. Configure the database connection in db/dbconn.php
  4. Run the project on a PHP-enabled web server
